DigiFarm UI Guidelines v1.0
Design system for all client-facing demos and internal tools. All components use the same tokens — dark, satellite-first aesthetic with agricultural green as the primary action color.

CSS Variables

Copy this block into every demo. All components reference these variables — never hardcode values.

/* DigiFarm Design System — v1.0 */ /* Copy into every demo :root block */ :root { /* Backgrounds */ --bg: #0E1621; /* main background */ --bg2: #17212B; /* panels, cards, sidebar */ --bg3: #1D2B39; /* elevated, hover rows */ --border: rgba(255,255,255,0.06); /* default border */ /* Brand */ --green: #009463; /* primary action */ --purple: #9C7FEF; /* secondary accent */ /* Text */ --white: #FFFFFF; /* primary text */ --muted: #8697A8; /* secondary text */ /* Semantic */ --success: #009463; --warning: #EA891D; --error: #F84A4A; --info: #4DB5FA; /* Shape */ --radius: 4px; /* universal border radius */ /* Typography */ --font: 'Inter', sans-serif; --mono: 'IBM Plex Mono', monospace; --text-base: 14px; --text-sm: 13px; --text-xs: 12px; --mono-size: 12px; /* captions, hints */ --h1: 20px; /* max headline size */ /* Map overlay blur */ --panel-bg: rgba(23,33,43,0.88); --panel-blur: blur(16px); }
